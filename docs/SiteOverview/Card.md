# Card
The **Card** component represents different games. 

[Site Overview](./SiteOverview.md)

## Props
### title
The name of the card.
> Type: string

### image
The image on the card.
> Type: string

### path
The path the card should navigate to. 
> Type: string

## Methods
### handleCardClick()
Navigates to the given page. 
> Args: path
